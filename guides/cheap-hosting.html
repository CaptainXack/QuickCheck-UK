How to host a small calculator site for free

Static sites are cheap because they do not need a database or server. QuickCheck UK can run from GitHub Pages, Cloudflare Pages, Netlify or Vercel.

  • GitHub Pages is simple if your code is already on GitHub.
  • Cloudflare Pages is fast and generous for static files.
  • Netlify and Vercel are also easy for small static sites.

Before adding ads, make sure the site has real pages, useful content, privacy information and working navigation.
