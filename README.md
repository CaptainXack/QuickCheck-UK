# QuickCheck UK

Fast everyday UK calculators and checkers for bills, rent, broadband, moving, TV, subscriptions and household costs.

This is a lightweight static site designed for free hosting and low running cost.

## What is included

- Mobile-first homepage
- 36+ useful calculators/checkers
- Search-style category layout
- AdSense placeholder zones
- Privacy, contact, about and advertising pages
- robots.txt, sitemap.xml and ads.txt placeholder
- GitHub Pages workflow
- No backend
- No database
- No paid API

## Edit AdSense

Replace the placeholder publisher ID in `ads.txt` and the ad blocks inside `index.html` / `assets/app.js` after your AdSense site is approved.

## Free hosting

This repo is ready for GitHub Pages, Cloudflare Pages, Netlify or Vercel.

For GitHub Pages, go to:

Settings > Pages > Build and deployment > GitHub Actions

Then the included workflow can publish the site.
